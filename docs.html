Tags

{exp:remember_me:set}

Save entry id’s to session storage. Session storage is cleared when the session expires. If no entry_id is specified, Remember Me will try to discover the currently viewed entry based on the last URL segment.

Parameters
entry_id
Either the entry id or the url title of the entry you want to save to session storage.
return
The URL to redirect to after the entry has been saved to storage. This should be in template_group/template format. If the current template has been called by via an Ajax call, the redirect is not performed.
Examples
{exp:remember_me:set}
{exp:remember_me:set entry_id=’45’}
{exp:remember_me:set entry_id=‘67’ return=‘template_group/template’}
